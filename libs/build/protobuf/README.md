# Google Protocol Buffer

Latest release [v3.6.1](https://github.com/google/protobuf/archive/v3.6.1.tar.gz)

## Build from Source

### Build in Linux and MacOS

requres: libtool autoconf shtool

```Bash
make -C protobuf-${VER}
```

### Build in Windows

```Powershell
build .\protobuf-$VER\vsprojects\protobuf.sln
```
